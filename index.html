Co to JAVA?

Java to najprostszy język obiektowy jaki znam. Jest on stworzony przez prostych ludzi dla prostych ludzi. Programowanie w nim to jak rozmowa z drugim człowiekiem! W dniu dzisiejszym jest to jeden z najbardziej potrzebnych języków w branży IT. Jednak jakie jeszcze java ma plusy?

Wiele Bibliotek

Java to język który ma tak wiele bibliotek że programiści mówią że człowiek nie żyje tak długo aby się ich wszystkich nauczyć. A pamiętajmy że wiele z nich jest prężnie rozwijana, więc jeżeli będziecie potrzebować zrobić cokolwiek to ogrom bibliotek pomoże wam w pisaniu mniejszej ilości kodu.

Prosta Budowa

Programowałem w wielu językach, c++, c#, java script, czy nawet python, ale najprostszą budową może się pościcić java. Nie używa się niepotrzebnych skrótów myślowych więc wszystko zostaje w głowie, dzięki czemu nauka ów języka jest bardzo prosta.
